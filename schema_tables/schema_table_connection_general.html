Option Type Required Description
args string X [DEPRECATED] Arguments that should be provided to the driver.
connection_type string X (Options described here)
driver string X [DEPRECATED] Name of driver class that should be used.
inputs array   One or more name(s) of model output(s) and/or new comm/file objects that the connection should receive messages from.
onexit string X Class method that should be called when a model that the connection interacts with exits, but before the connection driver is shut down. Defaults to None.
outputs array   One or more name(s) of model input(s) and/or new comm/file objects that the connection should send messages to.
translator array X Function or string specifying function that should be used to translate messages from the input communicator before passing them to the output communicator. If a string, the format should be “<package.module>:<function>” so that <function> can be imported from <package>. Defaults to None and messages are passed directly. This can also be a list of functions/strings that will be called on the messages in the order they are provided.
